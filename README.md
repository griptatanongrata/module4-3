// Loops //

_Definition_

-repeats the same code over and over again
-the key is to follow DRY (Don't Repeat Yourself)
-Types of Loops:
1: for loop
2: while loop
3: do while loop

_While_

-creates a looping statement with regards to fixed variables
-not used often

_For_

-creates a looping statement using specified variables and conditions
-used often
